# Features

## Current Features (MVP)

### Song Composition

- Input lyrics and select genre
- Auto-generate melody, harmony, drums
- Section-level regeneration for iteration
- ≤45s TTFP for preview

### Audio Production

- Voice synthesis from 5+ AI vocalist options
- Dynamic EQ and compression
- Stem isolation for mixing
- 48kHz/24-bit export quality

### Quality Assurance

- Similarity checking against reference audio
- Interval n-gram melody comparison
- Rhythmic DTW distance analysis
- Export gating with recommendations

### User Control

- Per-section regen without full re-composition
- Remix vibe from ≤30s reference audio
- Pro tier: budget slider for similarity
- Undo/redo for creative exploration

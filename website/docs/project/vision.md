# Project Vision

## Overview

Bluebird is an **AI music composition platform** that empowers musicians and creators to generate original, high-quality music from lyrics and creative direction.

## Mission

Lower barriers to music creation by combining AI composition, human creativity, and musical intelligence in a single platform.

## Core Values

- **Original Music Only** — No copyright infringement; all compositions are generated from scratch
- **Creative Control** — Users guide the AI with lyrics, style, and reference audio
- **Quality First** — Fast, high-fidelity output (≤45s TTFP) with export-time similarity checking
- **Trust & Transparency** — Clear similarity reports prevent accidental copyright issues

## Getting Started

```bash
# Clone and install
git clone https://github.com/MikeBlakeway/bluebird.git
cd bluebird
pnpm install

# Start development environment
pnpm dev

# Open in browser
# Web: http://localhost:3000
# API: http://localhost:4000
# Docs: http://localhost:3001 (this site)
```
